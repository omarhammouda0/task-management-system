üöÄ Quick Start

To use this API, follow these steps:

  1. Register a new account at /api/auth/register
  2. Login to get your JWT token at /api/auth/login
  3. Include the token in request headers: Authorization: Bearer YOUR_TOKEN
  4. Create Teams and Projects to organize your work
  5. Add Tasks to projects and assign them to team members
  6. Add Comments to tasks for discussions and updates
  7. Upload Attachments to tasks for file sharing
  8. Explore the Swagger UI for full documentation
